# Contributing to KeyboardTester.Click

Thank you for your interest in contributing to [KeyboardTester.Click](https://keyboardtester.click)!

## How to Contribute

### Report Bugs

Found a bug? Please let us know:

1. **Feedback Form:** [keyboardtester.click/feedback.php](https://keyboardtester.click/feedback.php)
2. **Email:** support@keyboardtester.click
3. **GitHub Issues:** Open an issue in this repository

### Suggest Features

Have an idea for a new tool or feature?

- Submit your suggestion at [keyboardtester.click/feedback.php](https://keyboardtester.click/feedback.php)
- Describe the feature and why it would be useful

### Translation Help

We're expanding to more languages! If you'd like to help translate:

Currently supported:
- [English](https://keyboardtester.click)
- [Spanish](https://keyboardtester.click/languages/spanish/)
- [French](https://keyboardtester.click/languages/french/)
- [German](https://keyboardtester.click/languages/german/)
- [Portuguese](https://keyboardtester.click/languages/portuguese/)
- [Russian](https://keyboardtester.click/languages/russian/)
- [Arabic](https://keyboardtester.click/languages/arabic/)
- [Japanese](https://keyboardtester.click/languages/japanese/)
- [Korean](https://keyboardtester.click/languages/korean/)

Contact us at support@keyboardtester.click to help with translations.

## Code of Conduct

- Be respectful and constructive
- Help others when possible
- Report bugs responsibly

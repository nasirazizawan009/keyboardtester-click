## 🎯 The Standard Pattern (All Pages Follow This)

Every tool page should now have this **5-part structure**:

```
1. HTML Head with ALL SEO Meta Tags
   ├─ Title (50-60 chars)
   ├─ Description (150-160 chars)  
   ├─ Keywords (5-8 terms)
   ├─ og:* tags (social media)
   ├─ JSON-LD structured data
   └─ Favicon

2. <?php include 'header.php'; ?>
   (Navigation, logo, theme toggle)

3. <main> with Tool Section
   ├─ <h1>Tool Name</h1> (ONLY ONE)
   ├─ Introduction paragraph
   ├─ Tool UI/Functionality
   └─ Images with alt text

4. Guidelines Section
   <?php include 'tools/[name]-guidelines.php'; ?>
   ├─ <h2>Help & Guidelines</h2>
   ├─ <h3>How to Use</h3>
   ├─ <h3>Features</h3>
   ├─ <h3>FAQs</h3>
   └─ <h3>Tips & Tricks</h3>

5. <?php include 'tools.php'; ?> + footer.php
   (Tools list + Footer)
```

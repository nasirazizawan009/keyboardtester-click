## 🎯 The Complete Pattern (5-Part Structure)

Every page now follows this proven pattern:

```php
┌─────────────────────────────────────────────────────────┐
│ 1. HTML Head (Complete)                                 │
│    - Title (50-60 chars)                               │
│    - Description (150-160 chars)                       │
│    - Keywords (5-8 terms)                              │
│    - og:* tags (social media)                          │
│    - JSON-LD structured data                           │
│    - Canonical URL                                     │
│    - Favicon link                                      │
└─────────────────────────────────────────────────────────┘
                         ↓
┌─────────────────────────────────────────────────────────┐
│ 2. <?php include 'header.php'; ?>                      │
│    (Shared navigation, logo, theme toggle)             │
└─────────────────────────────────────────────────────────┘
                         ↓
┌─────────────────────────────────────────────────────────┐
│ 3. <main> Tool Section                                 │
│    <h1>Tool Name</h1>                                  │
│    <p>Introduction</p>                                 │
│    Tool UI with images (all with alt text)             │
└─────────────────────────────────────────────────────────┘
                         ↓
┌─────────────────────────────────────────────────────────┐
│ 4. Guidelines Section (Modular)                        │
│    <?php include 'tools/[name]-guidelines.php'; ?>     │
│    - <h2>Help & Guidelines</h2>                        │
│    - <h3>How to Use</h3>                               │
│    - <h3>Features</h3>                                 │
│    - <h3>FAQs</h3>                                     │
│    - <h3>Tips & Tricks</h3>                            │
└─────────────────────────────────────────────────────────┘
                         ↓
┌─────────────────────────────────────────────────────────┐
│ 5. <?php include 'tools.php'; ?>                       │
│    <?php include 'footer.php'; ?>                      │
│    (Shared tools list + footer)                        │
└─────────────────────────────────────────────────────────┘
```

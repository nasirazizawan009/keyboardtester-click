# Security Policy

## About KeyboardTester.Click Security

At [KeyboardTester.Click](https://keyboardtester.click), we take security seriously.

### How Our Tools Work

All testing tools at [keyboardtester.click](https://keyboardtester.click) run **entirely in your browser**:

- **No data collection** - Your keystrokes, clicks, and inputs are never sent to our servers
- **No accounts required** - Use all tools without registration
- **No downloads** - Everything runs in the browser
- **HTTPS only** - All connections are encrypted

### Privacy

Read our full [Privacy Policy](https://keyboardtester.click/privacy-policy.php).

### Reporting Security Issues

If you discover a security vulnerability, please report it responsibly:

1. **Email:** support@keyboardtester.click
2. **Feedback Form:** [keyboardtester.click/feedback.php](https://keyboardtester.click/feedback.php)

Please do not disclose security issues publicly until we've had a chance to address them.

### Supported Tools

Security applies to all tools at [keyboardtester.click](https://keyboardtester.click):

- [Keyboard Tester](https://keyboardtester.click)
- [Mouse Test](https://keyboardtester.click/mouse-test.php)
- [Webcam Test](https://keyboardtester.click/webcamtesterindex.php)
- [Microphone Test](https://keyboardtester.click/mic-tester.php)
- [All Tools](https://keyboardtester.click/pages/tools.php)

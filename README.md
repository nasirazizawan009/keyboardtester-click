# KeyboardTester.click

A comprehensive online testing platform for keyboards, mice, audio devices, and various utility tools. Available in 9 languages with a modern, responsive design.

## 📁 Project Structure

```
keyboardtester.click/
├── index.php                    # Homepage
├── config.php                   # Site configuration
├── meta-config.php              # SEO meta configuration
│
├── assets/css/                  # Stylesheets
├── images/                      # Images (keyboard, mouse, shared)
├── includes/                    # Shared components
├── tools/                       # Tool components
├── languages/                   # 8 language versions
├── pages/                       # Category pages
├── blog/                        # Blog hub
├── help/                        # Help pages
├── api/                         # API endpoints
│
├── robots.txt                   # SEO crawl rules
├── sitemap-index.xml            # Sitemap index
└── *.xml                        # Individual sitemaps
```

## 💻 Technology Stack

| Category | Technology |
|----------|------------|
| Backend | PHP 7.4+ |
| Frontend | HTML5, CSS3, JavaScript (ES6+) |
| Styling | Custom CSS with CSS Variables |
| Icons | Font Awesome, Custom SVG |
| Fonts | Google Fonts (Space Grotesk, JetBrains Mono) |
| QR Codes | QRCode.js, jsQR |
| OCR | Tesseract.js |
| Graphics | Canvas API |

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🌙 **Dark/Light Theme** | Auto-detects system preference with manual toggle |
| 📱 **Responsive Design** | Mobile-first, works on all devices |
| ♿ **Accessibility** | ARIA labels, keyboard navigation, screen reader support |
| 🔍 **SEO Optimized** | Structured data, Open Graph, XML sitemaps |
| ⚡ **Performance** | WebP images, lazy loading, optimized assets |
| 🌐 **Multilingual** | 9 languages with native keyboard layouts |
| 🔒 **Privacy Focused** | All processing done locally in browser |
